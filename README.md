Asyncronous multiplayer experimentation backend for Arid

Copy .example.env to .env & add secrets.

## Python virtual env
```
    sudo apt install python3-venv -y
    python3 -m venv venv 
    source venv/bin/activate
```

## Python requirements
```
    pip install -r requirements.txt
```

## Run app (Dev only)
```
    python3 server.py
```
